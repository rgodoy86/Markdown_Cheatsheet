Most commonly used markdown.  

1. Headers

## 1. HEADERS
```
# TITLE H1  
## TITLE H2  
### TITLE H3  
#### TITLE H4  
##### TITLE H5  
###### TITLE H6  
```
# TITLE H1  
## TITLE H2  
### TITLE H3  
#### TITLE H4  
##### TITLE H5  
###### TITLE H6  

## 2. LISTS  
### Check List 
```
- [ ] Item   
- [X] Checked Item  
```
- [ ] Item   
- [X] Checked Item  

### Ordered List
```
1. Item 1  
1. Item 2  
1. Item 3  
1. Item 4  
```
1. Item 1  
1. Item 2  
1. Item 3  
1. Item 4  

### Unordered List
```
* Item with Asterisks  
+ Item with Plus  
- Item with Minus 
```
* Item with Asterisks  
+ Item with Plus  
- Item with Minus
